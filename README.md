## 🚀 Getting Started

### Prerequisites
```bash
node >= 14.0.0
npm >= 6.0.0
```

### Installation
1. Clone the repository
```bash
git clone https://github.com/WillAmbe-Corp/pro-portfolio-v1.git
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
